# Scraper API Project

Complete full-stack setup with FastAPI backend and React frontend.

## 🚀 Backend (FastAPI)

1.  **Navigate to root directory.**
2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the server:**
    ```bash
    python main.py
    ```
    *API will be available at `http://127.0.0.1:8000`*

## 🎨 Frontend (React + Vite)

1.  **Navigate to frontend directory:**
    ```bash
    cd frontend
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    *Frontend will be available at `http://localhost:3000`*

### 🛠 Tech Stack
-   **Backend**: FastAPI, Uvicorn
-   **Frontend**: React, Vite, Lucide Icons, Axios
-   **Styling**: Vanilla CSS (Premium Dark Mode)

### 📁 Project Structure
-   `app/`: Backend modules (auth, core, etc.)
-   `frontend/`: React application workspace
-   `main.py`: Entry point for the backend
-   `requirements.txt`: Python package list
